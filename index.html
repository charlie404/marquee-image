Complete Example with All Props

This example shows all available attributes with their values:

  • speed="100" - Animation speed (pixels per second)
  • desktop-margin="20" - Margin between images on desktop
  • tablet-margin="15" - Margin between images on tablet
  • mobile-margin="10" - Margin between images on mobile
  • desktop-height="72" - Height on desktop screens
  • tablet-height="54" - Height on tablet screens
  • mobile-height="36" - Height on mobile screens
  • repeat="10" - Number of times to repeat the image set
